Datasets#

The following datasets are useful for training, testing, and benchmarking neural networks on: common corruptions, out-of-distribution generalization tasks, and on other robustness-probing data.

CIFAR-Based Datasets#

CIFAR10C(root, corruption, severity[, ...])

A CIFAR-10-based test dataset for benchmarking neural network robustness to common corruptions [R2863170b71c2-1].

CIFAR100C(root, corruption, severity[, ...])

A CIFAR-100-based test dataset for benchmarking neural network robustness to common corruptions [Rb8cb5b192edb-1].

CIFAR10P1(root[, transform, ...])

CIFAR-10.1 dataset is a new test set for CIFAR-10.

ImageNet-Based Datasets#

ImageNet(root[, transform, ...])

ImageNetM10(root[, transform, ...])

ImageNet-M10, see https://arxiv.org/pdf/2112.15329.pdf

RestrictedImageNet(root[, transform, ...])

Restricted ImageNet, see https://arxiv.org/abs/1906.00945
